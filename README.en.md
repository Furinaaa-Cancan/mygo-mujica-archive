## About

This is a curated asset archive aimed at **fan creators, indie developers, and video editors** working with `MyGO!!!!!` and `Ave Mujica` content.

Official assets for these two bands are scattered across multiple sources (the anime site, Bushiroad Store, YouTube, Bestdori, Bandori Fandom). Filenames are inconsistent, duplicates abound, and folder structures don't match. This repo does the boring-but-useful work:

1. **Unified collection** — systematically gathered from primary official sources and reputable community sources
2. **md5 deduplication** — zero duplicates, including same-image-different-name across folders
3. **Renaming** — all files follow a consistent convention (see "Naming Convention" below)
4. **Symmetric taxonomy** — MyGO and Ave Mujica share an identical internal layout, making batch processing trivial
5. **Source verification** — every category is annotated with origin and copyright status

If you're building a fan site, character wiki, card browser, Discord bot, Live2D demo, fan-edit asset library, or wallpaper app — pick what you need and use it.

## Repository Structure

```
.
├── tachie/                            Character standees (transparent PNG)
│   ├── MyGO/                                 5 images,  1150 × 1741
│   └── AveMujica/                           10 images,   760 × 1160
│
└── assets/
    ├── _shared/                              Shared resources (KV / fonts / palette)
    │   ├── kv/                                 Official key visuals / hero art
    │   ├── fonts/                              5 web fonts (OFL, commercial-OK)
    │   └── palette/                            Character color palette (with CSS vars)
    │
    ├── MyGO/         brand · art · media · merch · wallpaper
    └── AveMujica/    brand · art · media · merch · wallpaper
```

Each band uses the same five sub-categories:

| Group | Contents | Use cases |
| :--- | :--- | :--- |
| `brand` | logos · SNS avatars · SNS banners | Branding, page headers, social cards |

## Naming Convention

```
Tachie / illustration   <index>_<JP name>_<romaji>[_suffix].png
                        e.g. 01_高松燈_Tomori.png
                             01_豊川祥子_Sakiko_面具版.png

Acrylic stand           <index>_<series>_<character>.jpg
                        e.g. 23_TwoMan_Tomori.jpg

MV thumbnail            <index>_<song>_<version>_<videoId>.jpg
                        e.g. 01_春日影_OfficialMV_ZsvJUh03MwI.jpg

Card                    <character>_res<id>_{normal|trained}_<title>.png
                        e.g. Anon_res037001_normal_Center_of_the_Band.png

Screenshot              Original Chinese scene names retained
                        e.g. (諸神的命運).jpg, [無詞]Mortis踢腿.jpg
```

Full index, source verification, and quality tiers: [`assets/README.md`](assets/README.md).

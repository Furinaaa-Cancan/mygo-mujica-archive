## 关于本仓库

这是一个面向**同人创作者、独立开发者、二次创作 UP 主**的素材整理库。

它存在的理由很简单——`MyGO!!!!!` 和 `Ave Mujica` 的官方素材分散在多个来源（动画官网、Bushiroad Store、YouTube、Bestdori、Bandori Fandom 等），命名混乱、重复严重、分类不一致。本仓库做了一件无聊但有用的事：

1. **统一收集**——从一手官方源 + 信誉社区源系统化抓取
2. **md5 去重**——全库 0 重复（包括跨目录的同图异名）
3. **重命名规范化**——所有文件按统一规则命名（详见下方"命名规则"）
4. **分类对称**——MyGO 与 Ave Mujica 内部目录结构完全一致，便于代码批处理
5. **来源核实**——每一类素材都标注来源与版权状态

如果你正在做：MyGO/Mujica 同人站、角色 wiki、卡组浏览器、Discord bot、Live2D 演示、二创视频素材库、桌面壁纸 App……都可以直接拿来用。

## 快速使用

### 单个文件
```
https://raw.githubusercontent.com/Furinaaa-Cancan/mygo-mujica-archive/main/<文件路径>
```

### 整库克隆
```bash
git clone https://github.com/Furinaaa-Cancan/mygo-mujica-archive.git
```

### 仅下载子目录
使用 [DownGit](https://downgit.github.io/)，粘贴目录链接即可打包下载 zip。

### 网页中嵌入示例
```html
<!-- 使用 jsDelivr CDN 加速（小文件适用） -->
<img src="https://cdn.jsdelivr.net/gh/Furinaaa-Cancan/mygo-mujica-archive@main/tachie/MyGO/01_高松燈_Tomori.png">

```

> 整库 ~466 MB。仅取部分素材时建议用 raw 链接 / DownGit / jsDelivr，避免完整克隆。

## 命名规则

```
立牌 / 立绘    <序号>_<日文姓名>_<罗马音>[_后缀].png
              例：01_高松燈_Tomori.png
                  01_豊川祥子_Sakiko_面具版.png

アクスタ       <序号>_<系列名>_<角色>.jpg
              例：23_TwoMan_Tomori.jpg

MV 缩略图     <序号>_<曲名>_<版本>_<videoId>.jpg
              例：01_春日影_OfficialMV_ZsvJUh03MwI.jpg

卡面          <角色英文>_res<id>_{normal|trained}_<卡名>.png
              例：Anon_res037001_normal_Center_of_the_Band.png

截图          保留原始中文场景命名
              例：(諸神的命運).jpg、[無詞]Mortis踢腿.jpg
```

完整索引、来源核实、质量分级见 → [`assets/README.md`](assets/README.md)

- 发现重复 / 错误命名 / 分类问题 → [开 Issue](https://github.com/Furinaaa-Cancan/mygo-mujica-archive/issues)
- 补充遗漏的素材（请遵守上方命名规则）→ 提交 PR
- 修订配色 hex / 字体许可信息 → 提交 PR
- 翻译 README（英 / 日）→ 欢迎

请勿提交：商业素材、付费内容截图、未经授权的二次创作。

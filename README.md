## 📦 仓库内容

```
.
├── tachie/                      ★ 角色立牌（透明 PNG）
│   ├── MyGO/                          5 张  1150×1741
│   └── AveMujica/                    10 张  760×1160（5 角色 × 面具/素颜）
│
└── assets/                      统一五大分组，详见 assets/README.md
    ├── _shared/                       共通（KV / 字体 / 配色）
    ├── MyGO/        → brand · art · media · merch · wallpaper
    └── AveMujica/   → brand · art · media · merch · wallpaper
```

每个项目内部统一使用 5 大分组：

| 分组 | 内容 | 适用场景 |
|---|---|---|
| `brand/` | logo / SNS 头像 / SNS 横幅 | 品牌识别、页面 header、SNS 卡片 |
| `art/` | 卡面立绘 / 投稿框 | 角色页主图、卡片网格 |
| `media/` | MV 缩略图 / 截图 / 精选名场面 | 影像合集、回忆画廊 |
| `merch/` | 官方アクスタ商品图 | 周边展示页 |
| `wallpaper/` | 桌面 / 手机壁纸 | 下载中心 |

## 🚀 快速使用

### 直接下载单个文件
GitHub 任意文件页右上角 **Download raw file**，或使用 raw 链接：
```
https://raw.githubusercontent.com/Furinaaa-Cancan/mygo-mujica-archive/main/<文件路径>
```

### 整库克隆
```bash
git clone https://github.com/Furinaaa-Cancan/mygo-mujica-archive.git
```

> ⚠️ 整库 ~466MB，仅需部分素材时建议用 raw 链接或 [DownGit](https://downgit.github.io/) 下载子目录。

### 在网站中引用
立牌 / KV / Logo 都已统一命名 + 透明背景 + 去重，可直接 `<img src>` 引用。

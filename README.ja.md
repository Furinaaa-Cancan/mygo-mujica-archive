## このリポジトリについて

これは **同人創作者・個人開発者・MAD/二次創作 UP 主** 向けに整理した素材アーカイブです。

`MyGO!!!!!` と `Ave Mujica` の公式素材は、アニメ公式サイト、Bushiroad ストア、YouTube、Bestdori、Bandori Fandom など複数の出所に分散しており、ファイル名・分類・重複の問題が常にありました。本リポジトリは地味ながら有用な作業を行いました：

1. **統一収集** — 一次公式ソース・信頼できるコミュニティソースから体系的に収集
2. **md5 重複排除** — クロスフォルダの同一画像も含め完全に 0 件
3. **命名規則統一** — すべて統一規則に従ってリネーム（後述「命名規則」参照）
4. **対称的な分類** — MyGO と Ave Mujica の内部構造を完全一致させ、バッチ処理を簡素化
5. **出所明記** — すべてのカテゴリーに出所と著作権ステータスを注記

同人サイト、キャラ Wiki、カード閲覧器、Discord ボット、Live2D デモ、二次創作素材庫、壁紙アプリなど、必要なものをそのままお使いください。

```
.
├── tachie/                            キャラクター立て看板（透過 PNG）
│   ├── MyGO/                                 5 枚  1150 × 1741
│   └── AveMujica/                           10 枚   760 × 1160

└── assets/
    ├── _shared/                              共通リソース（KV / フォント / 配色）
    │   ├── kv/                                 公式 KV / メインビジュアル
    │   ├── fonts/                              Web フォント 5 種（OFL · 商用可）

    │
    ├── MyGO/         brand · art · media · merch · wallpaper
    └── AveMujica/    brand · art · media · merch · wallpaper
```

各バンド内部は同一の 5 グループ構造：

| グループ | 内容 | 用途 |
| :--- | :--- | :--- |
| `brand` | ロゴ · SNS アイコン · SNS バナー | ブランディング、ヘッダー、SNS カード |
| `art` | カードイラスト · 投稿フレーム | キャラページ、カードグリッド |
| `media` | MV サムネ · スクショ · 名場面集 | 映像一覧、思い出ギャラリー |
| `merch` | 公式アクスタ商品画像 | グッズページ |
| `wallpaper` | デスクトップ · スマホ壁紙 | ダウンロードセンター、装飾背景 |

## 使い方

### 個別ファイル
```
https://raw.githubusercontent.com/Furinaaa-Cancan/mygo-mujica-archive/main/<ファイルパス>
```

### 全体クローン
```bash
git clone https://github.com/Furinaaa-Cancan/mygo-mujica-archive.git
```

### サブディレクトリのみ
[DownGit](https://downgit.github.io/) を使用し、フォルダ URL を貼り付けて zip ダウンロード。

### Web ページに埋め込む例
```html
<!-- jsDelivr CDN 経由（小さなファイル推奨） -->
<img src="https://cdn.jsdelivr.net/gh/Furinaaa-Cancan/mygo-mujica-archive@main/tachie/MyGO/01_高松燈_Tomori.png">

<!-- raw GitHub 直リンク（CDN なし） -->
<img src="https://raw.githubusercontent.com/Furinaaa-Cancan/mygo-mujica-archive/main/tachie/MyGO/01_高松燈_Tomori.png">
```

> 全体は約 466 MB あります。一部素材のみ必要な場合は raw / DownGit / jsDelivr の利用を推奨。

## 命名規則

```
立ち絵 / イラスト   <番号>_<日本語名>_<ローマ字>[_接尾].png
                  例：01_高松燈_Tomori.png
                      01_豊川祥子_Sakiko_面具版.png

アクスタ           <番号>_<シリーズ名>_<キャラ>.jpg
                  例：23_TwoMan_Tomori.jpg

MV サムネ         <番号>_<曲名>_<バージョン>_<videoId>.jpg
                  例：01_春日影_OfficialMV_ZsvJUh03MwI.jpg

カード            <キャラ英名>_res<id>_{normal|trained}_<カード名>.png
                  例：Anon_res037001_normal_Center_of_the_Band.png

スクショ           オリジナルの中国語シーン名を保持
                  例：(諸神的命運).jpg、[無詞]Mortis踢腿.jpg
```

完全索引・出所検証・品質階級は [`assets/README.md`](assets/README.md) を参照。

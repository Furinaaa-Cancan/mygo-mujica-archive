## 通用主题色（建站可用）

```css
:root {
  /* MyGO 主题 */
  --mygo-primary: #5A6FE0;     /* 燈，蓝紫 */
  --mygo-accent:  #FF8AB6;     /* 愛音，粉红 */
  --mygo-bg:      #F5F6FA;
  --mygo-ink:     #1B1C2E;

  /* Ave Mujica 主题 */
  --mujica-primary: #6B4F8C;    /* 祥子，紫 */
  --mujica-accent:  #C84B7E;    /* にゃむ，玫红 */
  --mujica-bg:      #15121C;    /* 暗紫黑 */
  --mujica-ink:     #EAEAF0;    /* Mortis 灰白 */
  --mujica-gold:    #C9A96B;    /* 哥特金边点缀 */
}
```
